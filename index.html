What Kind of game is N.O.V.A Legacy ?

N.O.V.A Legacy is an action filled fps sci-fi game with cool graphics and a variety of cool gears with an offline and online mode.We will be outlining some faqs about the game.

  1. How to get maxed out fast?

    2. One can get maxedout in less than 4months by coinfarming, playing multiplayer and events. coinfarming will generate more coins, events will return valuable cards and multiplayer will return some few rewards.

  2. What is coinfarming and how to do it ?

    3. Coinfarming is the process of getting coins faster in the game. It is done by using rocket launchers (advisably two) and going to the first mission of new ceres jungle and kill the enemies from the heliport. By doing this, you will obtain RPG kill (+50 coins ) for each successful shot.The image below explains better. For coin farming to be successful, its advisable to use boom and ignition as boom has the largest magazine size followed by ignition.Also, its advisable that boom should be upgraded to at least 220 and ignition 170.All these are from experience.
